## The Thing People Actually Get Excited About: Block Storage Slabs

If you spend time on hosting communities, you'll eventually notice that BuyVM regulars talk about "Slabs" with an unusual level of enthusiasm. Here's why.

Block Storage Slabs are separate NVMe-cached drives that you can attach to your VPS like an external hard drive — except they connect over InfiniBand RDMA, so in practice they behave like locally-attached storage. No I/O ceiling from the VPS, no throttling. Fast.
